What 'scope' is, and how it works in JavaScript.

Scope is used to define where variables are accessible from in a computer programme. In JavaScript there are two types of scope Global and Local.

Global scope variables are accessible from anywhere within the program. Global scope variables are defined outside of any function and can be read or changed from within any function.

Local scope variables are only accessible within the function that they are defined in. In functions where there is a local variable with the same name as a global variable the local variable will be used.

If a function is nested inside another function, the nested function has its own local scope plus the scope of the parent function.
