What is grid based design?

Grid based design is dividing a webpage into grids to place elements in desired place. A grid system is created with rows and columns, elements are nested inside these rows and columns.

Whats all the hype about responsive webpages?

These days the web is viewed on many different devices; Mobiles, tablets, laptops, desktops. These all have different screen resolutions. If a website is designed on a particular resolution screen, it may look fantastic on that screen but when it is opened on a different screen everything can be messed up. This can lead to a bad user experience.

Responsive design overcomes this problem, by designing the website to respond to the particular screen itâ€™s being displayed on. Responsive design can be implemented in many ways often in combination. This could be scaling an element with the screen size, altering the grid to change how many elements are displayed in a row, or not showing some less important elements altogether.

What is semantic structure?

Semantic structure is the naming of elements which are read by both the browser and the developer. They are used to identify and target specific elements. Semantic structure includes; tags, classes, ids. HTML5 includes a series of new semantic tags to make structure easier.
